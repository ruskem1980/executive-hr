================================================================================
  МАТРИЦА КОМПЕТЕНЦИЙ ТОП-МЕНЕДЖЕРОВ ПО РОЛЯМ
  Executive Competency Matrix — Role-Based Profiling
  Версия 3.0 | Axioma AI Executive Assessment System
================================================================================

ИНСТРУКЦИЯ: Матрица определяет приоритетные компетенции для каждой C-level роли.
Шкала важности: 5 = критическая, 4 = высокая, 3 = средняя, 2 = базовая, 1 = не важна.
Используется для: построения профиля, калибровки интервью, сравнения кандидатов.

Источники: Korn Ferry Leadership Architect, Spencer & Spencer, McKinsey Leadership Model,
Egon Zehnder, DDI, Lominger (FYI), Hogan Assessment.

================================================================================
  УНИВЕРСАЛЬНЫЕ КОМПЕТЕНЦИИ (обязательны для ВСЕХ C-level)
================================================================================

Компетенция              | Описание                                    | Мин. балл
-------------------------|---------------------------------------------|----------
Стратегическое мышление  | Видение на 3-5 лет, системный анализ        | 4/5
Лидерство                | Вдохновение, направление, accountability    | 4/5
Принятие решений         | Скорость, качество, в условиях неопределённ. | 4/5
Коммуникация             | Ясность, убедительность, адаптивность       | 4/5
Построение команды       | Найм, развитие, удержание талантов          | 4/5
Integrity                | Этичность, последовательность, доверие      | 5/5
Learning Agility         | Способность учиться из опыта, адаптироваться| 4/5
Самосознание             | Понимание своих сильных сторон и слабостей   | 4/5

================================================================================
  CEO — CHIEF EXECUTIVE OFFICER
================================================================================

Миссия роли: Определение стратегии, управление стейкхолдерами, обеспечение
устойчивого роста бизнеса, формирование культуры и команды лидеров.

Уровень Jaques: Stratum V-VII (в зависимости от масштаба компании)
Типичный PAEI: PaEI (стартап/рост) или pAEI (зрелая компания)

КОМПЕТЕНЦИЯ                      | ВАЖНОСТЬ | ПОВЕДЕНЧЕСКИЕ ИНДИКАТОРЫ
---------------------------------|----------|---------------------------------------
Визионерское лидерство           |   5/5    | Создаёт compelling vision, вдохновляет
                                 |          | организацию на амбициозные цели.
                                 |          | Балансирует long-term vision и short-
                                 |          | term execution.
Управление стейкхолдерами        |   5/5    | Эффективно работает с советом
                                 |          | директоров, инвесторами, регуляторами.
                                 |          | Управляет конфликтами интересов.
Стратегическое позиционирование  |   5/5    | Определяет конкурентную стратегию,
                                 |          | бизнес-модель, портфель продуктов.
                                 |          | Принимает bet-the-company решения.
Построение топ-команды           |   5/5    | Нанимает и развивает C-level лидеров.
                                 |          | Создаёт культуру accountability.
                                 |          | Умеет расставаться с неэффективными.
Управление культурой             |   5/5    | Формирует ценности, является ролевой
                                 |          | моделью. Культура как стратегический
                                 |          | актив.
Управление в кризисе             |   4/5    | Сохраняет ясность в турбулентности.
                                 |          | Быстро принимает решения с неполной
                                 |          | информацией. Anti-fragile.
P&L / Capital Allocation         |   4/5    | Понимает финансовые рычаги бизнеса.
                                 |          | Эффективно распределяет ресурсы.
Публичные коммуникации           |   4/5    | Представляет компанию вовне.
                                 |          | Media, конференции, investor relations.
M&A и партнёрства                |   3/5    | Оценивает и интегрирует сделки.
                                 |          | Стратегические альянсы.
Цифровая грамотность             |   3/5    | Понимает технологические тренды,
                                 |          | digital transformation, AI, data.

RED FLAGS для CEO:
- Микроменеджмент (не отпускает операционку)
- Нарциссизм (Hogan Bold на 90+ перцентиле без самосознания)
- История конфликтов с советами директоров
- Неспособность нанимать людей сильнее себя
- Избегание трудных разговоров и увольнений

================================================================================
  CFO — CHIEF FINANCIAL OFFICER
================================================================================

Миссия роли: Финансовая стратегия, управление рисками, обеспечение прозрачности,
поддержка принятия решений через данные, привлечение финансирования.

Уровень Jaques: Stratum IV-VI
Типичный PAEI: pAei (классический) или PAei (strategic CFO)

КОМПЕТЕНЦИЯ                      | ВАЖНОСТЬ | ПОВЕДЕНЧЕСКИЕ ИНДИКАТОРЫ
---------------------------------|----------|---------------------------------------
Финансовая стратегия             |   5/5    | Разрабатывает финансовую модель
                                 |          | бизнеса, стратегию фондирования,
                                 |          | структуру капитала.
Управление рисками               |   5/5    | Системный подход к идентификации,
                                 |          | оценке и митигации рисков.
                                 |          | Compliance, internal controls.
Аналитическое мышление           |   5/5    | Data-driven decision making,
                                 |          | финансовое моделирование, forecasting.
                                 |          | Видит паттерны в данных.
Операционная дисциплина          |   5/5    | Процессы, контроли, отчётность,
                                 |          | audit readiness. Точность и
                                 |          | своевременность.
Стратегическое партнёрство с CEO |   4/5    | Не просто учётчик, а стратегический
                                 |          | советник. Challenges CEO конструктивно.
                                 |          | Бизнес-чутьё помимо цифр.
Investor Relations               |   4/5    | Для публичных компаний — критична.
                                 |          | Коммуникация с рынком, аналитиками.
Управление казначейством         |   4/5    | Cash management, liquidity, working
                                 |          | capital optimization, FX hedging.
Digital Finance / FP&A           |   4/5    | Автоматизация финансов, BI,
                                 |          | real-time reporting, predictive analytics.
M&A Due Diligence                |   3/5    | Финансовый анализ сделок, integration
                                 |          | planning, synergies assessment.
ESG / Sustainability Reporting   |   3/5    | Растущее требование рынка и
                                 |          | регуляторов.

RED FLAGS для CFO:
- Чрезмерный консерватизм (блокирует рост из-за рисков)
- Отсутствие бизнес-чутья (видит только цифры, не стратегию)
- Слабые коммуникативные навыки (не может объяснить финансы не-финансистам)
- История проблем с аудитом или compliance
- Неготовность быть "плохим полицейским" (избегание конфронтации)

================================================================================
  COO — CHIEF OPERATING OFFICER
================================================================================

Миссия роли: Операционная эффективность, масштабирование, исполнение стратегии,
кросс-функциональная координация, построение системного управления.

Уровень Jaques: Stratum IV-V
Типичный PAEI: PAei (execution COO) или PaEi (transformation COO)

КОМПЕТЕНЦИЯ                      | ВАЖНОСТЬ | ПОВЕДЕНЧЕСКИЕ ИНДИКАТОРЫ
---------------------------------|----------|---------------------------------------
Операционное совершенство        |   5/5    | Строит масштабируемые процессы,
                                 |          | KPI-системы, continuous improvement.
                                 |          | Lean, Six Sigma, Agile в масштабе.
Execution и accountability       |   5/5    | Превращает стратегию в действие.
                                 |          | OKR/KPI каскадирование, tracking,
                                 |          | performance management.
Кросс-функциональное управление  |   5/5    | Координация между функциями.
                                 |          | Разрушение silos, alignment.
                                 |          | Influence без прямой власти.
Масштабирование                  |   5/5    | Опыт роста компании в 2-10x.
                                 |          | Процессы, которые работают на
                                 |          | следующем уровне масштаба.
Управление изменениями           |   4/5    | Change management, трансформация
                                 |          | операционной модели, digital
                                 |          | transformation.
Построение команды               |   4/5    | Структурирует организацию,
                                 |          | определяет роли, нанимает лидеров.
                                 |          | Развивает middle management.
Управление цепочками поставок    |   4/5    | Supply chain, logistics, vendor
                                 |          | management, procurement (для
                                 |          | производственных и ритейл компаний).
Технологическая грамотность      |   3/5    | Автоматизация, ERP, data analytics,
                                 |          | технологии для операционки.
P&L management                   |   3/5    | Управление себестоимостью, маржой,
                                 |          | операционными расходами.
Качество и безопасность          |   3/5    | Quality management, HSE, regulatory
                                 |          | compliance (для regulated industries).

RED FLAGS для COO:
- Перфекционизм без прагматизма (парализует организацию)
- Слабая коммуникация вверх (не умеет "продавать" результаты CEO)
- Микроменеджмент (не масштабируется)
- Отсутствие стратегического мышления (только тактика)
- Неспособность делегировать и доверять

================================================================================
  CTO — CHIEF TECHNOLOGY OFFICER
================================================================================

Миссия роли: Технологическая стратегия, архитектура, R&D, инновации, построение
и развитие инженерной команды и культуры.

Уровень Jaques: Stratum IV-VI
Типичный PAEI: PaEi (innovation CTO) или PAEi (scale CTO)

КОМПЕТЕНЦИЯ                      | ВАЖНОСТЬ | ПОВЕДЕНЧЕСКИЕ ИНДИКАТОРЫ
---------------------------------|----------|---------------------------------------
Технологическое видение          |   5/5    | Определяет tech strategy на 3-5 лет.
                                 |          | Выбор технологий, архитектуры,
                                 |          | платформ. Build vs buy decisions.
Инженерное лидерство             |   5/5    | Строит сильную инженерную команду
                                 |          | и культуру. Code review, standards,
                                 |          | development practices.
Инновации и R&D                  |   5/5    | Управление R&D pipeline. Баланс
                                 |          | innovation vs delivery. Patent
                                 |          | strategy (если применимо).
Архитектурное мышление           |   5/5    | Системный дизайн, scalability,
                                 |          | security, reliability. Technical
                                 |          | debt management.
Бизнес-ориентация                |   4/5    | Технология как enabler бизнеса,
                                 |          | а не как самоцель. ROI на
                                 |          | технологические инвестиции.
Управление delivery              |   4/5    | Agile at scale, DevOps, CI/CD.
                                 |          | Predictable delivery без снижения
                                 |          | качества.
Кибербезопасность                |   4/5    | Security by design, threat
                                 |          | modeling, incident response.
                                 |          | Compliance (GDPR, SOC2, ISO).
Управление техническим долгом    |   4/5    | Баланс speed vs quality.
                                 |          | Refactoring strategy, platform
                                 |          | modernization.
AI / Data Strategy               |   4/5    | ML/AI adoption, data platform,
                                 |          | analytics capabilities.
Vendor Management                |   3/5    | Cloud providers, SaaS tools,
                                 |          | outsourcing, contractors.

RED FLAGS для CTO:
- Технологический снобизм (over-engineering, "не изобретено здесь")
- Отсутствие бизнес-мышления (технология ради технологии)
- Неспособность объяснить технические решения бизнесу
- Устаревший tech stack и нежелание обновляться
- Слабое управление людьми (великий инженер ≠ великий лидер)
- Единая точка отказа (всё завязано на CTO лично)

================================================================================
  CMO — CHIEF MARKETING OFFICER
================================================================================

Миссия роли: Стратегия бренда и роста, customer acquisition, product marketing,
data-driven маркетинг, управление маркетинговым бюджетом и ROI.

Уровень Jaques: Stratum IV-V
Типичный PAEI: paEI (brand CMO) или PaEi (growth CMO)

КОМПЕТЕНЦИЯ                      | ВАЖНОСТЬ | ПОВЕДЕНЧЕСКИЕ ИНДИКАТОРЫ
---------------------------------|----------|---------------------------------------
Стратегия бренда                 |   5/5    | Brand positioning, storytelling,
                                 |          | репутация. Долгосрочное строительство
                                 |          | бренда vs тактический маркетинг.
Growth / Performance Marketing   |   5/5    | Customer acquisition, funnel
                                 |          | optimization, unit economics, CAC/LTV.
                                 |          | Digital marketing, SEO/SEM/SMM.
Customer Insight                 |   5/5    | Глубокое понимание клиента. Research,
                                 |          | segmentation, persona building,
                                 |          | voice of customer.
Data & Analytics                 |   5/5    | Marketing analytics, attribution,
                                 |          | A/B testing, predictive modeling.
                                 |          | Data-driven decision making.
Креативное лидерство             |   4/5    | Управление креативным процессом,
                                 |          | agencies, content strategy.
                                 |          | Баланс art & science.
Product Marketing                |   4/5    | Go-to-market strategy, positioning,
                                 |          | messaging, competitive intelligence.
                                 |          | Launch planning.
Revenue Partnership              |   4/5    | Alignment с Sales (RevOps).
                                 |          | Pipeline contribution, lead quality.
                                 |          | Маркетинг = revenue center.
PR и корпоративные коммуникации  |   3/5    | Media relations, crisis comms,
                                 |          | thought leadership, employer brand.
Международные рынки              |   3/5    | Локализация, cultural adaptation,
                                 |          | multi-market campaigns.
MarTech stack                    |   3/5    | CRM, CDP, automation, AI tools.
                                 |          | Технологическая зрелость маркетинга.

RED FLAGS для CMO:
- Креативность без метрик (не может показать ROI)
- Только digital или только brand (нужен full-stack)
- Нет опыта partnership с Sales (маркетинг в вакууме)
- Частая смена работ (средний tenure CMO — 2 года, если < 1.5 — red flag)
- Раздувание бюджетов без accountability

================================================================================
  CHRO — CHIEF HUMAN RESOURCES OFFICER
================================================================================

Миссия роли: People strategy, organizational design, talent management,
культура и вовлечённость, компенсации, HR-трансформация.

Уровень Jaques: Stratum IV-V
Типичный PAEI: paEI (strategic CHRO) или pAeI (operational CHRO)

КОМПЕТЕНЦИЯ                      | ВАЖНОСТЬ | ПОВЕДЕНЧЕСКИЕ ИНДИКАТОРЫ
---------------------------------|----------|---------------------------------------
Стратегическое HR-партнёрство    |   5/5    | HR как стратегическая функция.
                                 |          | Seat at the table. Влияние на
                                 |          | бизнес-решения через people lens.
Talent Management                |   5/5    | Talent acquisition, development,
                                 |          | succession planning, performance
                                 |          | management, talent reviews.
Организационное проектирование   |   5/5    | Org design, spans & layers,
                                 |          | operating model, restructuring.
                                 |          | Organizational effectiveness.
Культура и вовлечённость         |   5/5    | Culture design, employee engagement,
                                 |          | EVP, employer brand. Измерение и
                                 |          | управление культурой.
Change Management                |   4/5    | Управление трансформацией через
                                 |          | людей. Communication, training,
                                 |          | resistance management.
Compensation & Benefits          |   4/5    | Total rewards strategy, executive
                                 |          | compensation, equity programs,
                                 |          | market benchmarking.
HR Analytics                     |   4/5    | People analytics, predictive
                                 |          | modeling, workforce planning.
                                 |          | Data-driven HR.
Трудовое право и compliance      |   4/5    | Labor law, DE&I, ethics, employee
                                 |          | relations, union management (если
                                 |          | применимо).
Leadership Development           |   4/5    | Программы развития лидеров,
                                 |          | coaching, mentoring, high-potential
                                 |          | programs.
HR Technology                    |   3/5    | HRIS, ATS, LMS, employee experience
                                 |          | platforms, AI in HR.

RED FLAGS для CHRO:
- Административный HR (только кадровое делопроизводство)
- Неспособность challenge CEO и топ-команду
- Отсутствие бизнес-грамотности (не понимает P&L)
- Избыточная "мягкость" (избегание трудных решений по людям)
- Нет опыта работы с Board / Compensation Committee
- Фокус на процессах, а не на результатах

================================================================================
  ДОПОЛНИТЕЛЬНЫЕ РОЛИ (краткие профили)
================================================================================

CSO (Chief Strategy Officer):
  Ключевые: стратегический анализ (5), M&A (5), competitive intelligence (5),
  business development (4), board presentations (4).
  PAEI: paEi. Red flag: теоретик без execution.

CRO (Chief Revenue Officer):
  Ключевые: revenue growth (5), sales leadership (5), go-to-market (5),
  customer success (4), sales operations (4).
  PAEI: PAei. Red flag: "героический продавец" без системы.

CDO (Chief Digital Officer):
  Ключевые: digital transformation (5), technology vision (5),
  change management (4), innovation (5), stakeholder management (4).
  PAEI: PaEI. Red flag: technology-first без бизнес-результатов.

CPO (Chief Product Officer):
  Ключевые: product strategy (5), customer insight (5), UX/design thinking (4),
  data analytics (4), engineering partnership (4).
  PAEI: PaEi. Red flag: feature factory без стратегии.

CISO (Chief Information Security Officer):
  Ключевые: security architecture (5), risk management (5), compliance (5),
  incident response (4), security culture (4).
  PAEI: pAei. Red flag: "department of NO" без бизнес-enablement.

================================================================================
  КАК ИСПОЛЬЗОВАТЬ МАТРИЦУ
================================================================================

1. ПОСТРОЕНИЕ ПРОФИЛЯ:
   - Выберите роль → используйте соответствующий блок компетенций
   - Адаптируйте веса под стадию компании и конкретный контекст
   - Добавьте специфические отраслевые требования

2. КАЛИБРОВКА ИНТЕРВЬЮ:
   - Для каждой компетенции с весом 5 — минимум 2 STAR-вопроса
   - Для веса 4 — минимум 1 STAR-вопрос
   - Для веса 3 — можно проверить через referencing

3. СРАВНЕНИЕ КАНДИДАТОВ:
   - Используйте сводную таблицу для бенчмаркинга
   - Умножьте оценки кандидатов на веса
   - Сравните с "идеальным профилем"

4. РАЗВИТИЕ ДЕЙСТВУЮЩИХ ЛИДЕРОВ:
   - Gap-анализ: текущий уровень vs требуемый
   - Development plan на основе gaps
   - Coaching focus areas
